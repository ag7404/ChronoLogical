How to Use ChronoLogical

Chronological's interface is intuitive and user-friendly to use. However, the directions are listed below.

Add an Event

To add an event, fill out the time and event details inputs and then press the button labeled 'Add.' The entries will sort themselves according to time.

Delete an Event

To delete an event, hover over the table row and then click it. A dialog box will appear asking if you really want to delete the event. Click 'Yes' and the event will be removed.

Clear All Events

To remove all events, select the 'Clear' button. It will ask you to confirm your choice, before wiping the table clean of data.

Export your Schedule

If you would like to export your schedule for later use, select the 'Export' button. The contents of the table will be saved to your computer as a text file, which you can use by importing a schedule.

Import your Schedule

To import a file, you need to select the file first by clicking the button labeled 'Choose a File.' A file interface will appear, and you can select your file from there. After this, you need to click the 'Submit chosen file' button which will upload the contents of the file to your schedule. Any invalid files will not be accepted.
